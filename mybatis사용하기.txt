 1. 개념 (방법)
	1) vo 생성
		- vo 는 value object. 즉 그건 일반 클래스가 런타임때 자료를 담기위한. 
		  순수하게 value data 만을 담기 위한 클래스 
        - bean은 초기화시 직접 객체를 생성해서 컨테이너에 넣어놓고. 
		  사용자 요청에 따라 그 라이프 싸이클이 관리되는 모든 클래스나 객체
	2) mapper 생성 (sql 쿼리 + vo) → (dao, dto)
	3) config 설정 (jdbc + mapper) 
	4) java
